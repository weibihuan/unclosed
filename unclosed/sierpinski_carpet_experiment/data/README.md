# 数据目录说明

## 文件说明

### 1. 常数文件
- `constants.json` - JSON格式的常数数据（运行后生成）
- `constants.txt` - 文本格式的常数数据（运行后生成）

### 2. 连分数数据
- `cf_dimension_100_terms.txt` - 维数D的连分数前100项（运行后生成）
- `cf_dimension_500_terms.txt` - 维数D的连分数前500项（可选生成）

### 3. 参考数据
- `reference_constants.txt` - 参考常数数据（手动创建）

## 数据格式

### 连分数文件格式
维数D的连分数前100项
格式: 索引,值

0,1

1,1

2,8

3,3

4,18

...

### 常数文件格式
谢尔宾斯基地毯常数

维数 D = log₃8 = 1.892789260714372311298581...

面积 S = 8/9 = 0.8888888888888888888888889...

## 使用说明

这些数据文件由分析脚本自动生成，用于：
1. 存储计算结果
2. 后续分析使用
3. 结果验证
