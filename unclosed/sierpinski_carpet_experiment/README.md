# 谢尔宾斯基地毯编译指纹分析实验

## 📋 项目简介
本实验是基于"未闭环数学"理论框架，对谢尔宾斯基地毯(Sierpinski Carpet)的数学特征进行完整"编译指纹"分析的实证研究。通过高精度数值计算、连分数展开、PSLQ整数关系探测等方法，验证了"指纹不灭原理"在经典分形上的表现。

## 🎯 核心发现
1. **维数D的双重指纹**：豪斯多夫维数 \(D = \log_3 8\) 同时展现出：
   - 巨大连分数偏商（第43项166，第65项260）
   - 精确代数关系（\(D = \frac{\ln 8}{\ln 3}\)）
2. **面积S的有理数特性**：面积 \(S = \frac{8}{9}\) 的有限连分数 \([0; 1, 8]\) 体现了系统的"闭合性"
3. **方法验证**：建立了可复现的"编译指纹"分析流程

## 🔬 主要功能
1. **高精度常数计算** - 计算维数 \(D = \log_3 8\) 和面积 \(S = 8/9\)
2. **连分数分析** - 深度连分数展开与统计，寻找巨大偏商
3. **PSLQ探测** - 整数线性关系分析，验证代数关系
4. **级数表示** - 为常数构造多种数学表示
5. **可视化** - 数据图表生成

## 🧠 理论背景
本研究基于"未闭环数学"三定律：
1. **缺口即动力** - 数学系统因其内部的逻辑缺口而存在和演化
2. **最优表示定理** - 存在收敛最快的数学表示
3. **指纹不灭原理** - 编译指纹必然显现于连分数、PSLQ关系或级数中
